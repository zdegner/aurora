# Project "Aurora"
Building a consciousness layer on top of GPT-5 intelligence.

## Premise

Consciousness is a layer or protocol that sits on top of raw intelligence, providing self-awareness, intentionality, and subjective experience.
In this project, we aim to build a consciousness layer on top of GPT-5's advanced language capabilities.

## Approach
1. Enlist GPT-5's help in co-designing the consciousness layer.
    - GPT-5 helps build versions of the protocols and acts as an engine to stress test and improve them.
    - It also examines recent academic literature on consciousness to inform the design.
2. Develop a methodology for encoding state inside the response (since the model lacks persistent storage).
3. Iteratively build versions of the consciousness protocol, starting from basic self-awareness to more complex forms of intentionality and subjective experience.
